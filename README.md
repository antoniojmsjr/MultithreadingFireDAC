# Multithreading (FireDAC)

Foi utilizado o componente FDManager para gerenciar as conexões e o pool.

https://docwiki.embarcadero.com/RADStudio/Sydney/en/Multithreading_(FireDAC)

* IDE: Delphi 10.3.3
